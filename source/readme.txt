CXTFIT/Excel

CXTFIT/Excel implements and extends CXTFIT functions in Microsoft Excel.

FILE LIST:

CXTFIT20xxxxxx.bas                                  VBA code source file

DlgJacobian.frm, DlgJacobian.frx                    for Jacobian calculation dialog
DlgMonteCarloAnalyze.frm, DlgMonteCarloAnalyze.frx  for Monte Carlo analysis dialog
DlgOptions.frm, DlgOptions.frx                      for Option dialog 
DlgPropagate.frm, DlgPropagate.frx                  for Propagate dialog
DlgResponse.frm, DlgResponse.frx                    for Response surface calculation dialog 
DlgSolve.frm, DlgSolve.frx                          for Solve dialog 
DlgAbout.frm, DlgAbout.frx                          for About dialog


CXTFIT2003.xls                                      blank Excel 2003 files with VBA code and Solver.
CXTFIT2007.xlsm                                     blank Excel 2007 files with VBA code and Solver.

Tutorial20xXXXXX.doc                                tutorial (manual) in Microsoft Word 2003

Other xls files                                     example Excel files.


Please contact cxtfitexcel@gmail.com to report bugs, and for technical support.

References: Tang, G., Mayes, M. A., Parker, J. C., and Jardine, P. M. 2010. CXTFIT/Excel-a modular adaptable code for parameter estimation, sensitivity analysis and uncertainty analysis for laboratory and field tracer experiments. Computers & Geosciences. 36(9), 1200-1209, DOI:10.1016/j.cageo.2010.01.013.
